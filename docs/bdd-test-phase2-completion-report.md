### 2. 新创建的页面：管理员座位安排 ✅

**文件**: `web/src/app/admin/seats/page.tsx`

**功能特性**:
1. ✅ 考试选择
   - 只显示已关闭报名的考试（status='CLOSED'）
   - 单选框选择考试
   - 显示考试状态徽章

2. ✅ 分配策略选择
   - 5种分配策略：
     - `POSITION_FIRST_SUBMITTED_AT` - 岗位分组 + 报名时间（推荐）
     - `RANDOM` - 完全随机
     - `SUBMITTED_AT_FIRST` - 报名时间优先
     - `POSITION_FIRST_RANDOM` - 岗位分组 + 随机
     - `CUSTOM_GROUP` - 自定义分组
   - 推荐策略标记
   - 策略说明文字

3. ✅ 自定义分组支持
   - 当选择"自定义分组"策略时显示输入框
   - 可输入分组字段名称（如 graduationSchool, major, education）
   - 字段说明提示

4. ✅ 座位分配执行
   - 调用 `useAllocateSeats` hook
   - 参数验证（考试ID、自定义分组字段）
   - 加载状态显示
   - 成功/失败提示

5. ✅ UI/UX设计
   - 响应式布局（左侧考试选择，右侧策略选择）
   - 信息提示卡片（蓝色背景）
   - 选中状态高亮
   - 禁用状态处理
   - Toast通知

**API集成**:
- ✅ `useExams({ status: 'CLOSED' })` - 获取已关闭的考试列表
- ✅ `useAllocateSeats()` - 执行座位分配
- ✅ 错误处理 - `useErrorHandler()`
- ✅ Toast通知 - `useToast()`

**代码质量**:
- ✅ TypeScript类型安全
