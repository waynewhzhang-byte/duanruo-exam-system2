3. 已创建租户（例如：`duanruo-test2`）
4. 已登录租户管理员账号

- ⚠️ 旧入口：`/admin/exams/{examId}/form-config` 已废弃，会自动重定向到新入口

## 测试步骤

### 1. 创建考试并访问报名表单

**步骤：**
1. 访问 `http://localhost:3000/duanruo-test2/admin/exams`
2. 点击"创建考试"按钮
3. 填写考试基本信息（标题、代码、时间等）
4. 保存考试
5. 进入考试详情页面
6. 点击"报名表单"标签页

**预期结果：**
- ✅ 系统自动创建一个草稿状态的表单模板
- ✅ 显示"草稿"徽章

### 2. 编辑表单模板

**步骤：**
1. 在表单设计器中，点击"添加分区"
2. 从左侧字段库选择字段类型（例如：单行文本）
3. 点击字段添加到分区
4. 在右侧配置面板设置字段属性：
   - 字段名称：姓名
   - 字段键：name
   - 必填：是
   - 占位符：请输入您的姓名
5. 重复步骤 2-4，添加更多字段：
   - 邮箱字段（email）
   - 手机号字段（phone）
   - 文件上传字段（idCard）- 用于上传身份证
6. 点击"保存模板"按钮

**预期结果：**
- ✅ 字段成功添加到表单
- ✅ 字段属性正确保存
- ✅ 显示"表单模板保存成功"提示
- ✅ 表单状态仍为"草稿"

### 3. 预览表单

**步骤：**
1. 点击"预览"标签页

## 常见问题排查

### 问题1：表单模板未自动创建

**症状：** 访问"报名表单"标签页时，显示空白或加载中

**排查步骤：**
1. 检查浏览器控制台是否有错误
2. 检查 Network 标签页，查看 API 请求是否成功
3. 检查后端日志，查看是否有异常

**可能原因：**
- 租户信息缺失（`tenant?.id` 为 undefined）
- 后端 API 未实现或返回错误
- 权限不足

### 问题2：保存表单失败

**症状：** 点击"保存模板"后显示错误提示

**排查步骤：**
1. 检查浏览器控制台错误信息
2. 检查 Network 标签页，查看请求详情
3. 检查请求 payload 是否正确

**可能原因：**
- 字段数据格式不正确
- 缺少必填字段
- 租户ID未传递

### 问题3：发布表单失败

**症状：** 点击"确认发布"后显示错误提示

**排查步骤：**
1. 检查表单模板状态是否为 DRAFT
2. 检查后端日志，查看业务规则验证
3. 检查是否有字段验证错误

**可能原因：**
- 表单模板已经是 PUBLISHED 状态
- 表单模板没有字段（空表单）
- 后端业务规则限制

### 问题4：文件上传失败

**症状：** 上传身份证等附件时失败

**排查步骤：**
1. 检查 `fileupload.tsx` 组件是否传递了 `tenantId`
2. 检查 MinIO 服务是否运行
3. 检查后端文件服务配置

**可能原因：**
- 租户ID未传递到文件上传 API
- MinIO 服务未启动
- 文件大小超过限制

## API 端点参考

### 表单模板相关 API

```
GET    /api/v1/form-templates                      # 获取表单模板列表
POST   /api/v1/form-templates                      # 创建表单模板
GET    /api/v1/form-templates/{templateId}         # 获取表单模板详情
PUT    /api/v1/form-templates/{templateId}         # 更新表单模板
POST   /api/v1/form-templates/{templateId}/publish # 发布表单模板

GET    /api/v1/exams/{examId}/form-template        # 获取考试关联的表单模板
PUT    /api/v1/exams/{examId}/form-template/{templateId}  # 关联表单模板到考试
```

### 请求示例

**创建表单模板：**
```json
POST /api/v1/form-templates
Headers: X-Tenant-ID: {tenantId}

{
  "templateName": "考试报名表单 - 2ffc5a0b",
  "description": "自动创建的报名表单模板",
  "fields": []
}
```

**更新表单模板：**
```json
PUT /api/v1/form-templates/{templateId}
Headers: X-Tenant-ID: {tenantId}

{
  "templateName": "2024春季招聘考试报名表单",
  "description": "包含基本信息和附件上传",
  "fields": [
    {
      "id": "field-1",
      "fieldKey": "name",
      "fieldName": "姓名",
      "fieldType": "TEXT",
      "required": true,
      "placeholder": "请输入您的姓名",
      "order": 0
    },
    {
      "id": "field-2",
      "fieldKey": "idCard",
      "fieldName": "身份证扫描件",
      "fieldType": "FILE",
      "required": true,
      "order": 1
    }
  ]
}
```

**发布表单模板：**
```json
POST /api/v1/form-templates/{templateId}/publish
Headers: X-Tenant-ID: {tenantId}

{}
```
