## 预期问题和解决方案

### 1. 元素选择器问题
**问题**: 找不到页面元素
**解决**:
- 在world.ts中已实现智能fillField方法
- 支持多种选择器策略
- 使用重试机制

### 2. 异步等待问题
**问题**: 页面加载不完整
**解决**:
- 使用waitForLoadState
- 合理使用waitForTimeout
- 监听网络请求完成

### 3. Token获取问题
**问题**: 无法获取JWT token
**解决**:
- 从localStorage读取
- 从cookie读取
- 从API响应读取

### 4. 租户数据清理
**问题**: 测试数据残留
**解决**:
- 使用唯一时间戳
- AfterAll清理逻辑
- 租户soft delete
