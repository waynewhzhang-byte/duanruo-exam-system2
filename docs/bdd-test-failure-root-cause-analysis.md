### 分类2: API错误 (严重) 🟠

**影响场景数**: 8个 (14.3%)  
**严重程度**: 🔥🔥 **高**

#### 典型错误模式

```
[response 500] http://localhost:3000/api/v1/super-admin/tenants
[response 401] http://localhost:3000/api/session
[response 422] http://localhost:3000/api/v1/tenants/slug/undefined
```

#### 具体问题

1. **超级管理员API 500错误**
   - 场景: 超级管理员创建租户
   - 错误: `GET /api/v1/super-admin/tenants` 返回500
   - 原因: 后端API未实现或存在bug

2. **会话API 401错误** (频繁出现)
   - 错误: `GET /api/session` 返回401
   - 影响: 所有场景都有此错误日志
   - 原因: 会话管理机制问题，可能是正常的未登录状态检查

3. **租户Slug未定义**
   - 错误: `GET /api/v1/tenants/slug/undefined` 返回422
   - 原因: `testData.tenantSlug` 未正确初始化

#### 具体问题

   - 期望: "用户名已存在"
   - 实际: 可能是其他文本或未显示

   - 场景: 考生查看是否合格
   - 期望: 显示总分"263"

3. **标签页不存在**
   - 场景: 考生查看成绩统计图表
   - 期望: "统计分析"标签
   - 实际: 标签不存在或文本不匹配
