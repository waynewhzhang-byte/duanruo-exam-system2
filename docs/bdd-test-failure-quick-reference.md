## 🔧 常见问题修复模板

### 问题1: 找不到按钮

**错误**: `Error: 找不到按钮: 审核通过`

**修复**:
```typescript
<Button
  data-testid="btn-approve"
  onClick={handleApprove}
>
  审核通过
</Button>
```

---

### 问题2: API 500错误

**错误**: `[response 500] /api/v1/super-admin/tenants`

**修复**:
1. 检查后端Controller实现
2. 添加错误日志
3. 检查权限配置
4. 验证数据库查询

---

### 问题3: 元素超时

**错误**: `page.waitForSelector: Timeout exceeded`

**修复**:
1. 确认元素存在
2. 检查选择器正确性
3. 添加加载状态
4. 优化页面性能
